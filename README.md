# Hack-o-Week Projects Showcase

Welcome to my **Hack-o-Week** repository! This repository contains a collection of projects completed consecutively as part of a 19-week long challenge. The projects range from data analytics and machine learning to full-stack web applications and real-time backend systems.

## 🚀 Projects Overview

Here is a weekly breakdown of the projects implemented in this repository:

### ⚡ Energy & Sustainability Analytics (Weeks 1-10)
- **[Week 1: Peak Hour Electricity Spikes](./Week-1-Peak-Hour-Electricity-Spikes)** - Analyzing and predicting energy spikes during peak hours.
- **[Week 2: Classroom Usage Forecasting](./Week-2-Classroom-Usage-Forecasting)** - Forecasting room allocations and utilization to optimize energy.
- **[Week 3: Library Energy Forecast](./Week-3-Library-Energy-Forecast)** - Modeling energy consumption patterns in library facilities.
- **[Week 4: Cafeteria Load Prediction](./Week-4-Cafeteria-Load-Prediction)** - Predicting peak cafeteria loads for efficient resource management.
- **[Week 5: HVAC Optimization](./Week-5-HVAC-Optimization)** - Data-driven optimization of Heating, Ventilation, and Air Conditioning systems.
- **[Week 6: Sports Facility Usage](./Week-6-Sports-Facility-Usage)** - Analyzing utilization trends for sports complex energy savings.
- **[Week 7: Admin Building Analysis](./Week-7-Admin-Building-Analysis)** - Studying administrative building energy consumption profiles.
- **[Week 8: Parking Lot Lighting Forecast](./Week-8-Parking-Lot-Lighting-Forecast)** - Optimizing outdoor lighting based on usage and time series forecasting.
- **[Week 9: Hostel Laundry Peak Prediction](./Week-9-Hostel-Laundry-Peak-Prediction)** - Modeling peak laundry usage in campus hostels.
- **[Week 10: Campus Wide Sustainability Tracker](./Week-10-Campus-Wide-Sustainability-Tracker)** - A consolidated dashboard for tracking holistic campus sustainability metrics.

### 🌐 Secure Web & Real-Time Systems (Weeks 11-19)
- **[Week 11: User Registration Portal](./Week-11-User-Registration-Portal)** - Built a secure user registration and authentication framework.
- **[Week 12: Wearable Data Ingestion](./Week-12-Wearable-Data-Ingestion)** - Real-time ingestion service for wearable telemetry using WebSockets.
- **[Week 13: Dashboard Visualization](./Week-13-Dashboard-Visualization)** - Interactive front-end visualization for ingested health/telemetry data.
- **[Week 14: Data Encryption Pipeline](./Week-14-Data-Encryption-Pipeline)** - End-to-end data encryption and secure storage pipelines.
- **[Week 15: Basic Anomaly Detection](./Week-15-Basic-Anomaly-Detection)** - Implementation of algorithms to spot irregularities in data streams.
- **[Week 16: Real-Time Alert System](./Week-16-Real-Time-Alert-System)** - Event-driven architecture for generating real-time anomalies and alerts.
- **[Week 17: User Data Export Feature](./Week-17-User-Data-Export-Feature)** - Secure compliance feature allowing users to export their gathered data.
- **[Week 18: Admin Panel Setup](./Week-18-Admin-Panel-Setup)** - Advanced analytics and administrative control interface setup.
- **[Week 19: Admin Compliance Panel](./Week-19-Admin-Compliance-Panel)** - Complete administrative compliance and auditing interface.

## 🛠️ Tech Stack & Concepts

Throughout these 19 weeks, a variety of technologies and concepts have been utilized, including but not limited to:
- **Data Science/ML:** Python, Pandas, Machine Learning models (Forecasting, Anomaly Detection), Jupyter Notebooks
- **Backend Architecture:** Node.js, Express.js, WebSockets, REST APIs, Cryptography/Encryption
- **Frontend Development:** React, Vite, HTML/CSS/JavaScript
- **Databases & DevOps:** MongoDB, Git/GitHub, Environment Configuration
